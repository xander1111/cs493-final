## Requirements 

`Docker` - https://www.docker.com/

`NodeJS` - https://nodejs.org/en

`.env` - Environment file with mongoDB access credentials.

## Installing Dependencies

`npm install` - Installs all nodeJS modules needed.

## To Run (commandline)

`docker compose up` - Starts all the docker containers.

`sh test.sh` - Runs test script.

## Troubleshooting

`docker compose down -v` - Shuts down all docker containers.

`docker volume prune -f` - Removes unused docker volumes.

`docker compose up --build` - Rebuilds the containers and runs them.
